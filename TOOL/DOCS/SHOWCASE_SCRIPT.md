# Showcase script (~2 minutes)

Use tab **Demo** → button **Showcase ~15 giây** (gold). No API key required if PDF is on server.

## Before the room (5 min)

1. Open app URL in browser (Streamlit Cloud).
2. Tab **Demo** — confirm **Handbook mẫu** download works.
3. Optional: sidebar → paste API key only if you plan live AI demo.
4. Second browser tab on same **Demo** page as backup.

## Live flow (recommended)

| Time | Action |
|------|--------|
| 0:00 | Intro: "Trạm gửi tri thức — AI handbook for grade 6–9" |
| 0:20 | Tab **Demo** → enter topic → **Showcase ~15 giây** |
| 0:35 | Narrate steps 1→4 as progress bar moves |
| 0:50 | Click **Download Handbook** → show PDF |
| 1:00 | Tab **Quảng bá (Social)** → show Facebook posts |
| 1:20 | Tab **Chờ duyệt** / **Sách đã duyệt** if you want extra UI |
| 1:40 | Closing: mission + sample handbook for judges |

## If something fails

- **No API / key error** → stay on **Showcase ~15 giây** only.
- **No PDF** → use **Handbook mẫu: Đánh thức tư duy** button (right column).
- **App crash** → refresh backup tab.

## Do not use during showcase

- **Demo đầy đủ (có AI)** — 30s–2min, needs Gemini.
- **Xuất PDF** on new chapter — downloads images (slow).
- Generate new chapter from home unless you have working API + time.

## Fastest possible (10 sec)

1. Tab Demo.
2. **Handbook mẫu** download only — skip pipeline.
3. Say: "Full pipeline runs in 15s with Showcase button."
